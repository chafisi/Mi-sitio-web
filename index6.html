Técnicas Modernas de Diseño Visual (Sombras y Gradientes)

Ejemplos prácticos para crear profundidad, elevación y fondos ligeros.

1. Text-Shadow (Sombra de Texto)

La sombra de texto se usa para dar énfasis y "elevar" títulos, haciéndolos más legibles sobre fondos complejos o dándoles un efecto de profundidad suave.

TÍTULO CON SOMBRA FUERTE Y DEFINIDA

CSS: text-shadow: 1px 1px 3px rgba(0, 0, 0, 0.6);

2. Box-Shadow (Sombras Suaves / Elevación)

Las sombras de caja suaves elevan la percepción de un elemento, simulando que es una **capa** o una **tarjeta flotante** (efecto Neumorfismo o Skeumorfismo Suave).

Tarjeta Flotante

Esta caja tiene dos sombras combinadas para darle profundidad.

box-shadow: 0 10px 30px rgba(0,0,0,0.1), 0 2px 5px rgba(0,0,0,0.05);

3. Background Gradients

Los gradientes reemplazan las imágenes de fondo, siendo ligeros, adaptables a cualquier pantalla y modificables en tiempo real.

LINEAR GRADIENT (Banner)

El color cambia de izquierda (índigo) a derecha (púrpura). Ideal para banners largos o barras de navegación.

background: linear-gradient(to right, #4c51bf, #6b46c1);

El color irradia desde el centro hacia afuera, creando un efecto de "luz" o "foco".
background: radial-gradient(circle at center, #38bdf8, #0ea5e9);
