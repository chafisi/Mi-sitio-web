Diseño por Capas: Sombras y Gradientes

Tres ejemplos de tarjetas que simulan diferentes niveles de profundidad (Z-axis).

1. Elevación Suave

Simula una tarjeta o capa que está **ligeramente elevada** sobre el fondo. Ideal para elementos de navegación o información simple.

  • **Text-Shadow:** Suave para legibilidad.
  • **Box-Shadow:** Múltiples sombras claras.
  • **Gradiente:** Linear, casi imperceptible.

box-shadow: 0 10px 30px rgba(0,0,0,0.1), ...

background: linear-gradient(..., #ffffff, #f7f7f7);

2. Profundidad & Foco

Una capa más alejada que tiene su propia fuente de luz (el gradiente). Se usa para **banners** o elementos de llamada a la acción.

  • **Text-Shadow:** Glow blanco para destacar.
  • **Box-Shadow:** Oscura y proyectada (muy visible).
  • **Gradiente:** Linear diagonal (45deg) vibrante.

box-shadow: 0 20px 25px rgba(..., 0.4), ...

background: linear-gradient(45deg, #6b46c1, #4c51bf);

3. Efecto Presionado

Esta capa no se eleva, sino que parece **hundirse** en el fondo. El gradiente radial simula un punto de luz. Ideal para inputs o botones activos.

  • **Text-Shadow:** Blanco nítido.
  • **Box-Shadow:** Únicamente `inset` (sombra interior).
  • **Gradiente:** Radial (foco de luz central).

box-shadow: inset 0 2px 6px rgba(0,0,0,0.3), ...

background: radial-gradient(circle at center, #fef08a, ...);
