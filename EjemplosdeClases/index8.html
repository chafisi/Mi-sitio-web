Ejemplo de Layout con CSS Grid

El diseño cambia automáticamente al reducir el ancho del navegador.

Layout CSS Grid

  • **Desktop:** Header 100% arriba. Sidebar y Main Content se dividen el espacio de abajo (1/4 y 3/4).
  • **Móvil (Medida `md`):** Todos los elementos (`Header`, `Sidebar`, `Main`) se apilan uno debajo del otro y ocupan el 100% del ancho.
  • Se usan tres colores distintos para diferenciar las áreas: Indigo, Esmeralda y Ámbar.
