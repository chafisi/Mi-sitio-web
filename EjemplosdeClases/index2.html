Conceptos de Diseño Responsivo

Unidad 19 HCI - Mobile First, Rejillas Fluidas y Media Queries

1. Viewport Esencial (Base RWD)

Define que el ancho de la página es igual al ancho del dispositivo y la escala inicial es 1. **Sin esto, el RWD no funciona.**

<meta name="viewport" content="width=device-width, initial-scale=1.0">

2. Rejilla Fluida y Media Queries

Las columnas se apilan (100% ancho) en móvil (Mobile First), y se dividen (50%/50%) a partir de 768px (`min-width: 768px`).

Columna A (50% en escritorio)
Columna B (50% en escritorio)
/* Mobile First */
.col-fluid { width: 100%; }

/* Media Query (Breakpoint) */
@media (min-width: 768px) {
    .col-fluid {
        width: calc(50% - 7.5px); /* 50% menos el espacio entre ellas */
    }
}

3. Unidades Relativas (REM y EM)

Este texto usa **1.2rem** para su tamaño de fuente, escalando con la base del documento (Mobile First: 14px o 16px).

/* Tipografía Relativa: REM */
html { font-size: 16px; }
.text-rem { font-size: 1.2rem; }

/* Espaciado Relativo: EM */
.btn-em { padding: 1.5em 3em; } /* Relativo al font-size del botón (1rem) */

4. Imágenes Flexibles

Asegura que la imagen nunca desborde su contenedor, permitiendo que se adapte al tamaño de la columna donde se encuentre.

.flexible-image {
    max-width: 100%; /* La clave */
    height: auto;
}
