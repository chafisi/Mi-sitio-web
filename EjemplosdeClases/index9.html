Efecto Solitario con CSS Transform y Hover

Pasa el ratón por encima de la pila para ver cómo las cartas se deslizan y revelan.

CSS `transform` en Acción

  • Las tarjetas se posicionan con `position: absolute;` y tienen diferentes `z-index` (3, 2, 1) para el apilamiento.
  • El apilamiento inicial se logra con `transform: translate()` y `rotate()`.
  • El efecto de deslizamiento se aplica a **todas las tarjetas** cuando se hace `hover` en el contenedor padre (`.card-stack:hover`), lo que permite que se muevan de forma coordinada.

Tarjeta 1: Elevación Suave

Esta es la tarjeta superior. Al hacer hover en la pila, se desliza completamente a la izquierda para revelar las otras dos.

z-index: 3

box-shadow: 0 15px 30px...

Tarjeta 2: Profundidad Vibrante

La tarjeta del medio. Al hacer hover, se desliza a la mitad, quedando entre la primera y la tercera.

z-index: 2

background: linear-gradient(45deg...)

Tarjeta 3: Efecto Foco Radial

La tarjeta trasera. Al hacer hover, se reposiciona al centro, quedando totalmente visible para su lectura.

z-index: 1

background: radial-gradient(circle...)
